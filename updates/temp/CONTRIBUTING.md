# Contributing to Devalaya Pro

Thank you for your interest in contributing to Devalaya Pro! We welcome contributions from everyone.

## How to Contribute

1.  **Report Bugs**: If you find a bug, please open an issue on GitHub. Include details about your environment and steps to reproduce the bug.
2.  **Suggest Features**: Have an idea for a new feature? Open an issue to discuss it.
3.  **Submit Pull Requests**:
    *   Fork the repository.
    *   Create a new branch for your feature or bug fix.
    *   Write clear, commented code.
    *   Ensure your changes don't break existing functionality.
    *   Submit a pull request to the `main` branch.

## Development Setup

Follow the instructions in the [README.md](README.md) to set up your local development environment.

## Code Style

*   Follow PEP 8 for Python code.
*   Use semantic HTML and clean CSS.
*   Keep logic modular and well-documented.

## Code of Conduct

Please be respectful and professional in all interactions. We aim to build a welcoming community for everyone.
